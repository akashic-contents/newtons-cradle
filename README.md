# newtons-cradle

「ニュートンのゆりかご」のシミュレートです。
ボールは上部の空間に紐で固定されていて、固定位置を中心にドラッグで移動することができます。

## 実行方法

以下のコマンドで実行できます。

```
npm install
npm run build
npm run start
```
